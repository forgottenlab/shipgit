# ShipGit

ShipGit is a lightweight Git shipping assistant for safe commits, version tags, and multi-remote publishing.

ShipGit 是一个轻量级 Git 发布助手，适合需要同时维护 GitHub、Gitee 或其他远程仓库的开发者。

## Features

- Check Git project status
- Initialize Git repository
- Create recommended `.gitignore`
- Add and manage remotes
- Commit changes
- Create version tags
- Push branch and tag to multiple remotes
